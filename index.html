Some Things I've Built

theMovieGame

This is a movie guessing game. At the start of the game, a movie is randomly generated by the TMDb API. Each guess then provides clues by searching the API and finding if your guessed movie shares any actors, directors, or other similarities with the correct movie. This is a full stack application that utilizes an API I built using MongoDB to manage user settings.

  • React
  • Node.js
  • MongoDB
findMeASong.com

A react app that finds you a song based on your current mood. For this application, I created a song database that is mapped through to generate a random song for the user to listen to. I also created a version that implements the spotify API and uses authorization tokens to allow users access to a different playlist for each mood. (I have provided a demo video as Spotify limits user access until extension approval.)

  • React
  • Axios
  • Spotify API
myFreeBlackjack.com

In this blackjack game, the user is dealt two cards and plays against the dealer. The user has the ability to choose whether they want to recieve another card, or stick to their current card count. The deck is shuffled upon each new turn.

  • React
  • Deck of Cards API
reverbCalculator.com

This is a web app that helps music producers decide the reverb parameters for their track. The user simply inputs the track's BPM, and the app displays the ideal time for all the dimensions of the reverb to run. The user can then copy these values into their track's reverb settings.

  • HTML
  • CSS
  • JavaScript
Mason Murphy Music
